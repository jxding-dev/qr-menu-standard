# QR Mobile Menu Standard

카페 매장을 위한 QR 모바일 메뉴판 데모입니다.
고객이 테이블에서 메뉴를 확인하고, 옵션 선택 후 주문 내용을 복사하여 카카오톡 채널로 전달할 수 있는 구조입니다.

## 포함 기능

* 메뉴 20개 구성
* 카테고리 4개 분류
* 메뉴명 검색
* Best 메뉴 필터
* 매장 / 포장 주문 방식 선택
* 옵션 선택 모달
* 수량 변경 및 장바구니 총액 계산
* 장바구니 수량 조정 / 삭제
* 요청사항 입력
* 주문 내용 복사
* 카카오톡 채널 연결
* 전화, 인스타그램, 지도 링크 연결

## 파일 구조

```txt
qr-menu-standard/
  index.html    화면 구조
  style.css     모바일 UI 디자인
  app.js        장바구니, 옵션, 검색, 주문 로직
  menuData.js   매장 정보와 메뉴 데이터
```

## 수정 위치

고객 매장에 맞춰 변경할 내용은 대부분 `menuData.js`에서 수정합니다.

* `name`: 매장명
* `tableName`: 테이블명 또는 지점명
* `kakaoUrl`: 카카오톡 채널 URL
* `categories`: 메뉴 카테고리
* `commonOptions`: 공통 옵션 묶음
* `menus`: 메뉴명, 설명, 가격, 사진, 옵션

## 구성 기준

스탠다드 패키지 기준 구성입니다.

* 메뉴 20개 이하
* 카테고리 4개 이하
* 옵션 선택 및 총액 계산
* 주문 내용 복사
* 카카오톡 채널 연결
* 모바일 화면 최적화
* 간단한 매장 정보 링크 연결

## 사용 방법

1. 프로젝트 파일 다운로드
2. `index.html` 실행
