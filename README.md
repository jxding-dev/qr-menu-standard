# QR Menu Standard

스탠다드 패키지용 카페 QR 모바일 메뉴판 데모입니다.

## 포함 기능

- 메뉴 20개, 카테고리 4개
- 메뉴 검색
- Best 메뉴 필터
- 매장/포장 주문 방식 선택
- 옵션 선택 모달
- 수량 변경과 장바구니 총액 계산
- 장바구니 수량 조정/삭제
- 요청사항 입력
- 주문 내용 자동 복사 후 카카오톡 채널 열기
- 전화, 인스타그램, 지도 링크

## 파일 구조

```text
qr-menu-standard/
  index.html    화면 구조
  style.css     모바일 UI 디자인
  app.js        장바구니, 옵션, 검색, 주문 로직
  menuData.js   매장 정보와 메뉴 데이터
```

## 수정 위치

고객 매장에 맞춰 바꿀 내용은 대부분 `menuData.js`에서 수정합니다.

- `name`: 매장명
- `tableName`: 테이블명 또는 지점명
- `kakaoUrl`: 카카오톡 채널 URL
- `categories`: 메뉴 카테고리
- `commonOptions`: 공통 옵션 묶음
- `menus`: 메뉴명, 설명, 가격, 사진, 옵션

## 판매 기준

스탠다드 권장 구성:

- 메뉴 20개 이하
- 카테고리 4개 이하
- 옵션 선택과 총액 계산
- 카카오톡 주문 내용 복사
- 모바일 최적화
- 간단한 매장 정보 링크 연결
